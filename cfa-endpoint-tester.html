🖼 Test 1 — Text to Image (no reference images)
Use this when you want to generate an image from a prompt alone — no character photos, no scene photos.

Endpoint: fal-ai/kling-image/v3/text-to-image
Required: Prompt only. Write what you want to see. The model will generate purely from your description.
🎨 Test 2 — Image with Reference Slots (the main workflow path)
This mirrors the full CFA image generation workflow. There are four reference inputs:

  • Subject — Upload a photo of the primary character. Image upload only. Always becomes @Image1 in your prompt.
  • Co-Subject — Upload a second character if two people appear. Image upload only. Becomes @Image2 if uploaded.
  • Scene — Either upload a photo of the environment, or switch to ✏ Describe and type it out. If an image is uploaded, it gets an @ImageN number. If text is used, it is automatically appended to the prompt as Setting: ...
  • Style — Same as Scene. Upload a visual reference image, or switch to ✏ Describe and write the style. Text mode appends Style: ... to the prompt automatically.

The @Image numbering is automatic. As you upload images, each slot gets assigned a number in this order: Subject → Co-Subject → Scene → Style. The live legend below the slots always shows you which number belongs to which slot before you write your prompt.

Endpoint auto-selects:
· If any images are uploaded → fal-ai/kling-image/o1 (character/style reference model)
· If only text descriptions are used → fal-ai/kling-image/v3/text-to-image
✍ Writing the Prompt for Test 2
When using the o1 reference model, your prompt must tell the model how to use each uploaded image. Reference them using the @ImageN tags shown in the live legend.

Example: If Subject is @Image1 and Scene is @Image2, write:
"Draw @Image1 sitting in the environment from @Image2, cinematic lighting, 16:9"
If you are using text descriptions for Scene and Style instead of images, you do not need to reference them in the prompt — they are automatically added for you.
🎬 Test 3 — Video from Reference Images
Generates a video clip using Kling's image-to-video model.

Endpoint: fal-ai/kling-video/v1.6/pro/image-to-video
  • Start Frame (required) — Upload the image you want to be the very first frame of the video. This locks the opening visual so your character doesn't drift between clips.
  • Subject (optional) — Upload a character reference to keep identity consistent throughout the clip.
  • Prompt — Describe the motion, camera move, and mood. No @Image tags needed here.
  • Duration — 5 or 10 seconds. Video generation takes approximately 60–90 seconds.
